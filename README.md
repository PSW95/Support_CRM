# Support CRM System

A full-stack customer support ticketing system built with FastAPI, SQLite, and Tailwind CSS.

## Features

-  Create support tickets with customer info
-  List all tickets with clean interface
-  Search tickets across all fields
-  Filter by status (Open, In Progress, Closed)
-  View ticket details
-  Update ticket status
-  Add notes/comments to tickets
-  Auto-generated ticket IDs
-  Mobile responsive design

## Tech Stack

- **Backend**: Python + FastAPI
- **Database**: SQLite (SQLAlchemy ORM)
- **Frontend**: HTML + Tailwind CSS + Vanilla JS
- **Deployment**: Railway.app
